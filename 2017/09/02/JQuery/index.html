#JQuery

###一.基础和介绍

1.   怎么使用jQuery？
    1.先引入jQuery的文件
    2.入口函数(例如onload，jQuery里面叫ready)
    3.调用jQuery提供的方法
2. 版本
    1. .x版本：兼容老版本浏览器，例如IE6，7，8.现在jQuery作者已经不更新1.x的功能了，但是呢会优化和解决部分bug

    2. x版本：抛弃了老版本浏览器，只兼容新式浏览器。新增了部分新的方法。停止更新了

    3. x版本：抛弃了老版本浏览器，只兼容新式浏览器。新增部分新的方法，目前会更新
4. Jquery其实就是一套js的库文件；

###二.

1. onload和ready的对比：
    1. window.onload方法 ，
        1. 如果window.onload用了多个，那么后面的会覆盖前面的
         2. onload是页面加载完毕，包括dom元素还有外部文件    
    2. ready方法(入口函数)
        1. 如果有多个，不会覆盖，而是会整合到一起，比onload方法执行的时间要早
        2. ready触发的时间：是所有dom元素都创建完毕就触发
    3. $(document.body)|$('body')
2. $()函数介绍  $===jquery
    1. 就是jquery里面定义的函数
    2. 作用：可以把dom元素来一个变身，变成能力更强大的jQuery对象，那么就有jQuery里面写好的方法了，如果要调用jQuery给我们提供的更好的方法，就需要用$()给某个dom元素进行包装
    3. 完整写法：window.$
    4. 最完整写法：
        1. window.jQuery(document.body).css('background', 'hotpink');
        2. jQuery(document.body).css('background', 'hotpink');
    5. 如果需要同时写多个属性赋值，那么css里要传入一个对象{}
    6. jq绑定事件，只要对某个jq对象.事件名（不用加on）
    7. jQuery里面不用循环，直接设置就可以了。因为它内部会帮我们循环，并且给每个元素设置成我们写的css，隐式迭代，jQuery的特点：有隐式迭代，自动帮我们循环
    8. 
        1. 伪类选择器
            1. $('input:first')
            2. $('input:eq(1)')代表找到下标为1的input标签（下标从0开始）
            3. $('input:last')
        2. 过滤有关的方法
            1. $('input').first().click
            2. $('input').eq(1).click
            3.  $('input').last().click
        4.   $('li:even')偶数
        5.    $('li:odd')奇数
    9. 如果要使用jQuery的方法，还得把this包装起来$(this)

16. on绑定事件
    1. 
                用法一：
                        jquery对象.on(事件字符串,function);
                用法二：
                        jquery对象.on("事件1 事件2 事件3 ………………",function);
                        代表所有事件都调用同一个function
                用法三：
                        jquery对象.on({

                            '事件1':方法1,
                            '事件2':方法2,
                            '事件3':方法3
                            ......
                        });
                        给一个对象绑定多个事件，并且每个事件有不同的响应方法            
                用法四：
                        jquery对象.on(事件字符串,选择器,function);

                        首先要在jquery对象范围内触发事件，并且还要满足选择器才能触发
                        例如'li'：这就代表在jquery对象范围内触发，并且还要是它的子元素为li标签的触发

                        注意：事件字符串里可以写多个事件，代表多个事件也绑定同一个方法                
                用法五：
                        jquery对象.on({
                            '事件1':方法1,
                            '事件2':方法2,
                            '事件3':方法3
                        },选择器);

###

1. attr取checked时，很可能取到undefine
2.  什么时候得到的不是undefine，就是当checkbox在标签内写一个checked属性时，但是此时，用attr取，无论如何取到的结果都是checked
3.  如果要取checked状态最正确的值，那么通过attr是拿不到的，用prop才拿得到, prop取是正常的
4.  动画队列
    1.  写动画效果，其实就是把这些效果加在动画队列里面，一个一个按顺序的执行
    2.  队列：先进先出，所以按执行顺序写，这样就不会
        1.  $('.box').animate({
                top:100,
                left:100
            },500)///////这样会同步执行
        2. 这样是执行完top后再继续执行left
            $('.box').animate({
                top:100,
            },500)
            $('.box').animate({
                left:100
            },500)
     3. 如果你通过queue方法加入了一个任务到队列，那么原来队列后面的内容默认不会执行，如果要执行必须再调用dequeue的方法    
             1.  $('.box').queue(function () {
                        console.log('我加入到队列里了');
                        $('.box').dequeue();
                    })  
5. 使用插件：
    1. jquery的颜色过渡效果插件
        1. 顺序必须是先导入jQuery，再导入jQuery框架
        2. 此框架的作用就是让animate也支持backgroundColor

    2. jquery的颜色过渡效果插件
        1. 一定要先导入jQuery，再导入jQueryUI
        2. css一定放在根目录，然后引入这个css文件
        3. 去demo里喜欢什么效果，就扒它的代码拿过来就行了
        4. <script src="js/jquery-ui.js"></script>
             <link rel="stylesheet" href="jquery-ui.css">
        5. 为了保证jQueryUI的各种功能都正常，css和js文件都要导入，除此以外，image文件夹也要拖到项目 
6. 多库共存
    1. 假如说以后你项目里导入了另外一个库，$也是那一个库的关键字
    2. 那么就把jQuery的$符号作用去掉就行了
    3. var r = $.noConflict();///这个方法也会有一个返回
    4. 值，你用什么变量接收，那么这个变量名就相当于原来的$
